## 🚀 Features

* **Real-time Analysis:** Processes video/image feeds with ultra-low latency directly to mobile devices.
* **State-of-the-Art AI:** Utilizes **YOLOv10** for highly accurate, next-generation object detection and classification.
* **Native Android Experience:** A sleek, highly responsive mobile application built entirely in **Kotlin**.
* **Scalable Architecture:** Designed with modern, modular code allowing for seamless edge-device integration.
